#### 1. Create a terraform.tfvars file with your info:

```
tenancy_ocid = "YOUR TENANCY OCID"

user_ocid = "YOUR USER OCID"

fingerprint = "YOUR FINGERPRINT"

private_key_path = "YOUR PRIVATE KEY PATH"

region = "YOUR REGION"
```
#### 2. Run the Terraform

```
terraform init
terraform validate
terraform plan
terraform apply
```
